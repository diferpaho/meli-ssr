# meli-ssr

## Cloning the repository

In your terminal run the command:

```` 
git clone https://github.com/diferpaho/meli-ssr.git
````

```` 
cd meli-ssr
````

### Install dependencies


```` 
npm install
````


### Running the project


```` 
npm run dev
````
